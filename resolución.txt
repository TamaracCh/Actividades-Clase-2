5_ el cambio realizado en la interfaz web no está localmente porque desde github Desktop tenemos que recargar cambios del repositorio remoto tocando Fetch para que se actualicen las modificaciones, y luego poder hacer pull para que se guarden dichos cambios finalmente en el espacio local.

8_ Cuando intentamos hacer un push de los cambios aparece un conflicto ya que se está modificando el mismo archivo desde el repositorio remoto y desde el repositorio local al mismo tiempo.
